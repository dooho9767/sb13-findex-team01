# 김도형 구현 기능 상세

(자신이 개발한 기능에 대한 사진이나 gif 파일 첨부)

## 지수 데이터 관리 API

지수 데이터 등록, 수정, 삭제 API 구현

일자별 지수 데이터를 저장하기 위한 `IndexData` Entity 및 Repository 구현

지수 데이터 생성/수정 요청을 처리하는 Service 계층 로직 구현

지수 데이터 삭제 API를 구현하고, 연관 데이터 삭제 흐름을 물리 삭제 기준으로 정리

## 지수 데이터 검증 및 타입 관리

지수 데이터 입력값의 음수 검증 등 요청 데이터 유효성 검사 보완

`indexType`을 문자열 대신 Enum으로 관리하도록 변경하여 타입 안정성 개선

지수 데이터 삭제 및 연관 데이터 삭제 흐름 정리

## 지수 데이터 Swagger 및 예외 처리

지수 데이터 API Swagger 문서화

지수 데이터 Controller 오류 수정 및 API 응답 오류 보완

지수 데이터 도메인 예외와 전역 예외 처리 연동
